What is Bundletool

Bundletool is an official Google tool used to work with Android App Bundles (.aab). It allows generating and installing split APKs on a device.

Download Bundletool

Download bundletool from the official GitHub releases page.

Download Bundletool

Download the file named bundletool-all-*.jar.

Configure Bundletool in ADB Deck

  1. Download the file bundletool-all-*.jar
  2. Save it to any convenient folder
  3. Open Settings → Bundletool in ADB Deck
  4. Click Select Bundletool
  5. Select the bundletool-all-*.jar file
  6. Press Check to verify that the tool works

Note

Bundletool requires a Java Runtime Environment (JRE). Java is usually already installed on most systems.
