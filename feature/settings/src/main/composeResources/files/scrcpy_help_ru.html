Что такое scrcpy

scrcpy — это инструмент с открытым исходным кодом, который позволяет отображать экран Android-устройства на компьютере и управлять устройством с помощью мыши и клавиатуры.

Скачать scrcpy

Скачайте scrcpy с официальной страницы проекта:

Download scrcpy

Выберите архив для вашей операционной системы и распакуйте его.

Настройка в ADB Deck

  1. Скачайте scrcpy
  2. Распакуйте архив
  3. Откройте Settings → scrcpy в ADB Deck
  4. Нажмите Select scrcpy
  5. Выберите исполняемый файл scrcpy
  6. Нажмите кнопку Check, чтобы проверить запуск

Где находится файл scrcpy

Windows

  • scrcpy.exe внутри распакованного архива

macOS

  • /opt/homebrew/bin/scrcpy
  • /usr/local/bin/scrcpy

Linux

  • /usr/bin/scrcpy
  • /usr/local/bin/scrcpy

В настройках ADB Deck необходимо выбрать именно этот файл.
