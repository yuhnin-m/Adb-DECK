Что такое ADB

ADB (Android Debug Bridge) — официальный инструмент Google для подключения компьютера к Android-устройству. Он распространяется в составе пакета Android Platform Tools.

Скачать ADB

Скачайте Platform Tools с официального сайта Google.

Download Platform Tools

После загрузки распакуйте архив в любую папку.

Настройка в ADB Deck

  1. Скачайте архив Platform Tools
  2. Распакуйте архив
  3. Откройте Settings → ADB в ADB Deck
  4. Нажмите Select ADB
  5. Выберите файл adb из папки platform-tools
  6. Нажмите кнопку Check, чтобы проверить ADB

Где искать файл adb

Если у вас установлен Android Studio, ADB обычно уже находится в системе.

macOS

  • ~/Library/Android/sdk/platform-tools/adb
  • ~/Android/Sdk/platform-tools/adb
  • /opt/homebrew/bin/adb
  • /usr/local/bin/adb

Linux

  • ~/Android/Sdk/platform-tools/adb
  • ~/Android/sdk/platform-tools/adb
  • /usr/bin/adb
  • /usr/local/bin/adb

Windows

  • C:\Users\USERNAME\AppData\Local\Android\Sdk\platform-tools\adb.exe
  • C:\Android\sdk\platform-tools\adb.exe
  • C:\Program Files\Android\Android Studio\platform-tools\adb.exe
