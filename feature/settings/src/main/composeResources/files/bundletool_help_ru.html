Что такое Bundletool

Bundletool — это официальный инструмент Google для работы с Android App Bundle (.aab). Он позволяет собирать и устанавливать split APK на устройство.

Скачать Bundletool

Скачайте bundletool с официальной страницы GitHub:

Download Bundletool

Скачайте файл bundletool-all-*.jar.

Настройка в ADB Deck

  1. Скачайте файл bundletool-all-*.jar
  2. Сохраните его в удобную папку
  3. Откройте Settings → Bundletool в ADB Deck
  4. Нажмите Select Bundletool
  5. Выберите файл bundletool-all-*.jar
  6. Нажмите кнопку Check, чтобы проверить инструмент

Примечание

Bundletool требует установленную Java Runtime Environment (JRE). На большинстве систем Java уже установлена.
