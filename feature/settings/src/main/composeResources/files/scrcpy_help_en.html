What is scrcpy

scrcpy is an open-source tool that allows you to mirror and control an Android device from your computer using a keyboard and mouse.

Download scrcpy

Download the archive for your operating system and extract it.

Configure scrcpy in ADB Deck

  1. Download scrcpy
  2. Extract the archive
  3. Open Settings → scrcpy in ADB Deck
  4. Click Select scrcpy
  5. Select the scrcpy executable file
  6. Press Check to verify that scrcpy runs

Where to find the scrcpy executable

Windows

  • scrcpy.exe inside the extracted archive

macOS

  • /opt/homebrew/bin/scrcpy
  • /usr/local/bin/scrcpy

Linux

  • /usr/bin/scrcpy
  • /usr/local/bin/scrcpy

Select this executable in ADB Deck settings.
