What is ADB

ADB (Android Debug Bridge) is an official Google tool used to connect a computer to an Android device. It is distributed as part of the Android Platform Tools.

Download ADB

Download Platform Tools from the official Google website.

Download Platform Tools

After downloading, extract the archive to any folder.

Configure ADB in ADB Deck

  1. Download the Platform Tools archive
  2. Extract the archive
  3. Open Settings → ADB in ADB Deck
  4. Click Select ADB
  5. Select the adb file inside the platform-tools folder
  6. Press Check to verify that ADB works

Where to find the adb file

If Android Studio is installed, ADB usually already exists on your system.

macOS

  • ~/Library/Android/sdk/platform-tools/adb
  • ~/Android/Sdk/platform-tools/adb
  • /opt/homebrew/bin/adb
  • /usr/local/bin/adb

Linux

  • ~/Android/Sdk/platform-tools/adb
  • ~/Android/sdk/platform-tools/adb
  • /usr/bin/adb
  • /usr/local/bin/adb

Windows

  • C:\Users\USERNAME\AppData\Local\Android\Sdk\platform-tools\adb.exe
  • C:\Android\sdk\platform-tools\adb.exe
  • C:\Program Files\Android\Android Studio\platform-tools\adb.exe
